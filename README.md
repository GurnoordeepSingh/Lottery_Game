## 🎮 Game Description

There will be 5 players.
Each player starts with 100 virtual coins.
The entry fee for a game can vary (starting from 10 coins up to 100 coins).
The system will generate a set of lottery tickets, including winning tickets.
Players will choose one ticket from the available set.
The system will lock the chosen ticket to each player.
The winning tickets will be revealed, and the prize pool will be distributed among the winners based on their chosen ticket:
1st place: 50% of the prize pool
2nd place: 75% of the remaining prize pool
3rd place: The rest of the remaining prize pool
4th and 5th place: No winnings (if they chose a losing ticket)
After each round, player coin balances will be updated, and the game can be reset for a new round with new tickets.
